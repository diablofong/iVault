iVault

Privacy Policy

Last updated: April 2026

No data collection

iVault does not collect, transmit, or store any personal data. All photo transfers happen entirely on your local machine over USB. Nothing is sent to external servers.

What iVault stores locally

A configuration file is saved at ~/.ivault/config.json on your computer. It contains your selected backup folder path and backup history (file counts, timestamps). This file never leaves your device.

Network access

iVault makes no network requests during normal operation. The app communicates with your iPhone exclusively over USB using Apple's AFC protocol — no Wi-Fi, no internet connection required.

Third-party services

This website (GitHub Pages) is hosted by GitHub. GitHub may collect standard server logs (IP address, browser type) as described in GitHub's Privacy Statement.

Open source

iVault is fully open source. You can inspect exactly what the app does at github.com/diablofong/iVault.
