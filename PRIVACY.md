# Privacy Policy

**iVault does not collect, transmit, or store any personal data.**

## What iVault does

iVault transfers photos and videos from your iPhone to your Windows PC over a local USB connection using Apple's AFC protocol. All file transfers happen entirely on your own hardware — no data is sent to any server, cloud service, or third party.

## What iVault stores locally

iVault writes a single configuration file to `%USERPROFILE%\.ivault\config.json` on your PC. This file contains:

- The backup destination folder path you chose
- Backup history (file counts, timestamps)
- Resume state for interrupted backups (file count, total)

This file never leaves your device.

## What iVault does NOT do

- Does not upload photos or any file content to the internet
- Does not collect analytics, crash reports, or usage telemetry
- Does not require an account, login, or email address
- Does not use cookies or tracking identifiers
- Does not access the internet at all during normal operation

## GitHub Pages (this website)

The iVault website is hosted on GitHub Pages. GitHub may collect standard web server logs (IP address, browser, page visited) as described in [GitHub's Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement). iVault itself has no control over and no access to these logs.
